# 用户手册

## 功能概览
- 自定义倒计时时间（分钟 + 秒）
- 开始计时
- 暂停/继续
- 重置
- 倒计时结束弹窗与声音提醒
- 可选 wav/mp3 文件提醒音
- 可选循环播放闹钟
- 可设置提醒音量（0-100%）
- 系统托盘最小化运行
- 表盘与指针联动显示（分针/秒针随倒计时实时更新）
- 到时弹窗提醒（含振动闹钟 SVG 图标）
- 快捷定时按钮（1 个历史定时 + 5 个自定义定时）
- 主题模式切换（跟随系统 / 浅色 / 深色）
- 音频设置区域默认折叠，可手动显示/收起

## 使用步骤
1. 在“分钟/秒”输入框设置目标时间。
2. 点击“开始”进入倒计时。
3. 中央表盘会同步显示倒计时进度，秒针每秒走动，分针随剩余时间变化。
4. 可使用快捷定时按钮：
	- 预设区顶部有快捷提示：左键应用、右键保存、拖拽交换。
	- “历史定时”：点击后可从最近使用的定时列表中选择并应用。
	- “自定义1~5”：左键应用；右键弹出菜单可选择“保存当前定时 / 设置名称 / 取消”；按住左键拖拽到另一个自定义按钮可交换两者预设。
5. 需要中断时点击“暂停”，再次点击“继续”。
6. 点击“重置”可停止并回到输入时间。
7. 计时到 00:00 时，会弹出“时间到”窗口，并显示振动的闹钟图标；同时执行声音提醒。
8. 如需设置提醒音，先点击“显示音频设置”展开音频区。
9. 勾选“音频提醒”，点击“选择音乐”选择本地 `wav/mp3` 文件（默认打开 `C:/Windows/Media`）。
10. “音量”区域显示当前百分比；点击数值按钮会弹出竖直音量条，可拖动或滚轮调节；点击界面其他位置后自动收起。
11. 如需到时持续提醒，可勾选“循环”；点击弹窗“确定”后停止循环。
12. “播放/暂停”测试按钮同时显示图标和文字：播放态为“播放图标 + 播放”，播放中为“暂停图标 + 暂停”。
13. 音频设置可再次收起，不影响已保存设置。
14. 点击窗口最小化或关闭按钮时，应用会隐藏到系统托盘继续运行。
15. 右上角提供两个主题按钮：
	- “跟随系统”开关（苹果风格）：打开后界面跟随 Windows 当前明暗主题自动切换。
	- Emoji 主题按钮（☀️/🌙）：仅在“跟随系统”关闭时可点击，用于在浅色与深色之间切换，图标会随当前主题变化。
	- 主题控件与“分钟/秒”输入位于同一行；将鼠标悬停在“跟随系统”开关上可查看当前状态提示。

## 注意事项
- 输入时间需大于 0，否则无法开始。
- 计时过程中输入框会锁定，避免误改。
- 未选择有效音频文件时，将使用系统蜂鸣提示。
- 点击音频测试前请先选择有效的 `wav/mp3` 文件。
- 托盘菜单中可“打开”主界面或“退出”应用。
- 若检测到已有番茄钟实例正在运行，会提示“已有任务在运行”，并在 5 秒倒计时后自动退出当前新开的实例。

## 配置文件
- 程序会自动将上次选择的音频路径、启用状态与主题模式保存到 `config/settings.ini`。
- 预设定时（含名称）与历史定时也会保存到配置文件。
- 若配置文件不存在，会在首次保存时自动创建。
