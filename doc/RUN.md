# 编译/运行说明

## 运行方式
### Qt Creator
- 构建完成后，点击“运行”启动 `TomatoClock`。

### 命令行
- 可执行文件一般位于：
  - `build/TomatoClock.exe`（单配置生成器）
  - 或 `build/<Config>/TomatoClock.exe`（多配置生成器）

## 运行校验
- 应用启动后应显示：
  - 分钟与秒输入框
  - 中央时间与圆形倒计时表盘
  - 快捷定时按钮区（历史 + 自定义）
  - 状态文本
  - 操作按钮行（开始、暂停、重置、退出）
  - “显示音频设置”切换按钮（默认折叠音频区）
- 点击“显示音频设置”后应出现：
  - 音频提醒、循环开关
  - 音量按钮（点击弹出竖向滑条）
  - 选择音乐、文件名、播放/暂停测试按钮
- 到时应出现“时间到”弹窗并执行声音提醒（或蜂鸣回退）。

## 启动行为
- 程序采用单实例运行：若检测到已有实例，会提示并在倒计时后退出新实例。
- 最小化/关闭窗口时默认进入系统托盘继续运行。
